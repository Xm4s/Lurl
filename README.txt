Lurl - Local URL

ABOUT
Lurl is PHP app to make an http request and view the response.
The idea is to have a portable app for testing API on your local machine or in a private network.

REQUIREMENTS
- PHP 5

INSTALLATION
Just copy this folder anywhere on your web server and then access it via browser.
